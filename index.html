Tentang Saya

Saya adalah seorang mahasiswa Teknik Informatika di Universitas Indraprasta PGRI yang memiliki minat dan pengalaman kuat dalam pengembangan aplikasi web. Saya terbiasa membangun sistem informasi berbasis PHP native dan MySQL, mulai dari aplikasi absensi dan penggajian berbasis QR code, sistem perpustakaan, hingga aplikasi kasir dan manajemen gudang. Dalam pengembangan proyek, saya menguasai konsep dasar hingga lanjutan seperti login multi-peran, manajemen session, validasi form, dan pengelolaan data menggunakan query SQL yang kompleks.

saya juga menguasai konsep dan implementasi Object-Oriented Programming (OOP), baik dalam pembuatan class, pemisahan logika, maupun struktur proyek agar lebih rapi dan modular. Selain pengembangan back-end, saya mampu membuat antarmuka pengguna yang sederhana dan fungsional dengan HTML, CSS, JavaScript, serta memanfaatkan framework seperti Bootstrap dan library seperti SweetAlert untuk meningkatkan pengalaman pengguna. Saya memiliki semangat belajar tinggi, mampu bekerja mandiri maupun dalam tim, dan selalu tertarik mengeksplorasi teknologi baru yang dapat mendukung efisiensi pengembangan aplikasi.
